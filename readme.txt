这个项目用于一站式管理服务器，要使用这个文件需要安装node.js，服务器默认控制台管理密码是admin和password，需要在控制台输入chcp 65001，才会成为兼容网页utf8的格式，否则看不清命令了个啥，这个js开了3个端口，分别负责投票系统，文件管理系统，和命令系统，可以自己编写网页，也可以使用里面的index模板在里面更改超链接到服务器的地址
This project is used for one-stop management of the server, to use this file you need to install node .js, the default console management password of the server is admin and password, you need to enter chcp 65001 in the console, it will become a compatible web page UTF8 format, otherwise you can't see what the command is, this js opens 3 ports, respectively responsible for the voting system, file management system, and command system, you can write your own web page, You can also use the index template inside to change the address of the hyperlink to the server
